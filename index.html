How to use?

Firstly power the installation to a socket.After that the system will connect automatically to the local WI-FI.After that access this page.Make sure the STATUS is ON(green color) on top right.


Now you can Edit the variables as you see fit for making a PID regulator with your values, add a duraction and a sample time.After that simply press the Send Data button and the data you entered will be sent to the system and the motor will start.


On results page you will see the stability of the system,on error page the system error, finally if you want to see the exact values head over to the console page.
